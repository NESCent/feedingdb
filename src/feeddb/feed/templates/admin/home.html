This database collects physiologic data and associated metadata for a number of mammalian species. The data contain information on muscle activity, bone and muscle strain, jaw and oropharyngeal apparatus motion, and intra-oral pressure and were generated using several techniques (e.g., electromyography, cineradiography, sonomicrometry).

The purpose of collecting data of this type is to address fundamental questions about the evolution, functional morphology, physiology and development of the mammalian head. The data are searchable and can be downloaded into csv format.
