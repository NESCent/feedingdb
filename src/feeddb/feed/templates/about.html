About FEED

{% endblock %} {% block content %}

The Feeding Experiments End-User Database (FEED) was designed and developed by the Mammalian Feeding Apparatus Working Group supported by the National Evolutionary Synthesis Center (NESCent).

Technology

  • Database management system: PostgreSQL 8.3
  • Programming language: python
  • Web framework: Django (1.1.1)
  • Other libraries
    • South for schema and data migration
    • django-pagination (1.0.5)
    • jQuery
    • psycopg2 (2.0.13): PostgreSQL adapter for the Python programming language

Credits

  • Concept and scientific requirements: NESCent Working Group on Analysis and Synthesis of Physiologic Data from the Mammalian Feeding Apparatus, led by
    • Christine E. Wall (Duke University)
