About

Data

This database contains data and meta data of EMG, kinematic, and bone strain for mammalian species that have been collected by the working group participants over many years. The purpose of collecting data of this type is to address fundamental questions about the evolution, functional morphology, and development of the mammalian head. The data are searchable and can be downloaded into csv format.

About the demo site and example database

To illustrate how the system work, we created a demo site with an example dataset, representing a subset of the actual database. Please visit our demo site for details
